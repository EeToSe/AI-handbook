# 网络访问与连接要求

本页说明课题组成员使用订阅制 AI 服务时的网络连接要求。网络配置应保持稳定，避免触发异常登录、频繁验证或账号限制。

## 1. 连接要求

1. 仅使用课题组确认的访问方式和网络配置。
2. 使用客户端时优先选择规则模式，避免全局代理。
3. 不要频繁切换地区、节点、IP、设备或浏览器环境。
4. 更换网络节点、设备或浏览器前，应先退出相关账号；切换完成后再重新登录。

## 2. 使用限制

1. 网络服务仅用于访问课题组批准的 AI 工具和必要的学术研究服务。
2. 不得用于下载、爬虫、视频、游戏、个人娱乐或其他大流量用途。
3. 不得将课题组网络配置转发给课题组之外的人员。
